# commander

完整的 [node.js](http://nodejs.org) 命令行解决方案。

[官方文档](https://github.com/tj/commander.js/blob/HEAD/Readme_zh-CN.md)介绍的很详细，本文的**主要目的**是介绍常规的使用方式，也便于平常的快速阅读与查找。

## 约定

- 必填，使用尖括号 `<>`；
- 选填，使用方括号 `[]`；

## 优点

- 支持自动生成 help 信息；

## 缺点

- 必填 required 是不做检查的，需要开发者自己去检查用户有没有传必填参数；
